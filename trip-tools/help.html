🚀 Trip Tools — Help

Add Dispatch Details

  • Select a cell in column A to set the target row
  • Paste dispatch text into the dialog
  • Conflicts with existing cell values are shown before writing
  • Pickup/drop cities are written to columns D and E
  • Schedule time slots fill columns J–W based on pickup date
  • Drive time and total miles are written two columns past the last schedule column

Update Trip Status

  • Select a cell to set where Time and Distance are written
  • Enter current city — press Enter or click Calculate
  • Adjust destination (defaults to first drop) and speed (defaults to 50 mph)
  • Time is written to the selected cell; Distance one row below
  • A timestamped note (time, city, user) is prepended to the Time cell on each update
  • "Did you mean?" hint appears if the city name may be misspelled
